# African-Car-Sales-Dashboard
An interactive Excel Dashboard that analyzes African car sales trends, revenue, customer distribution and top-performing car models using Pivot Tables, Pivot Charts, Slicers and KPI cards.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## Key Insights
- Car sold in each Country across the year
- Units sold by Car model
- Top 3 City by Sales
- Top 5 selling car Model
- Revenue by Car make
- Sales by Car type
- Sales by Country
